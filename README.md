# Decision Log App

A simple decision-making log application built with Next.js and localStorage.

## Features

- Create decisions with context and options
- View all decisions in a clean list format
- Delete decisions
- Persistent storage using localStorage

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
npm install
```

### Running the App

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- **app/** - Next.js app directory with pages and layouts
- **components/** - Reusable React components
- **styles/** - CSS styling files
- **lib/** - Utility functions (storage helpers)
- **public/** - Static assets

## Usage

1. Navigate to the home page to view all decisions
2. Click "Add Decision" to create a new decision
3. Fill in the decision details and submit
4. Delete decisions from the list view as needed

## Technologies

- Next.js 14
- React 18
- localStorage API
- CSS3
